# Visual Studio Code Extension - Planning Syntax Language

## Overview

A Visual Studio Code extension to support intellisense and color in text file `(*.pln)` 
to organize Epic, Backlog, Sprint, Story, Task.

## Snippet Code

* Keyword: 
    * Epic
    * Backlog
    * Sprint
    * Story
    * Task
* List
    * team
    * developer
    * now
    * date
